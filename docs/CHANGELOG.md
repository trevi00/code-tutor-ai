## [1.4.0] - 2025-01-05

### Phase 4: AI 튜터 고도화 및 대회 수준 문제

#### Added
- **AI 튜터 7단계 힌트 시스템**
  - 문제 이해 → 개념 연결 → 아이디어 유도 → 의사코드 → 구조 설계 → 코드 작성 → 복잡도 분석
  - 단계별 이모지 아이콘 (📋📎💡📝🏗️✍️📊)
  
- **코드 리뷰 강화**
  - 시간/공간 복잡도 자동 분석
  - O(n²) → O(n) 최적화 제안
  - 코드 스타일 개선 제안
  
- **디버깅 도움 시스템**
  - IndexError, RecursionError 등 오류 패턴 분석
  - 원인 분석 및 해결 가이드 제공
  - 엣지 케이스 식별

- **문제 추천 시스템 (WeaknessAnalyzer)**
  - 약점 패턴 분석
  - 난이도별 맞춤 추천
  - 학습 진도 기반 추천

- **대회 수준 문제 25개 추가**
  - 코드포스 Div2 스타일 (8문제): Segment Tree, 상태 압축 DP, 트리 DP 등
  - ICPC/IOI 스타일 (6문제): Convex Hull Trick, FFT, Max Flow 등
  - 삼성 SW 역량테스트 (6문제): 복잡한 시뮬레이션, 완전탐색 최적화
  - 카카오/네이버 코테 (5문제): 문자열 파싱, 좌표 압축

- **알고리즘 패턴 18개 추가** (총 45개)
  - 고급 자료구조: Sparse Table, Sqrt Decomposition, Persistent Segment Tree
  - 그래프: Bellman-Ford, Floyd-Warshall, Articulation Bridges, 2-SAT
  - DP: LIS O(n log n), Bitmask DP, Interval DP, Digit DP
  - 문자열: Rabin-Karp, Z-Algorithm, Aho-Corasick, Manacher

- **받아쓰기 고급 템플릿**
  - 세그먼트 트리 구간합 (30줄)
  - 펜윅 트리 BIT (19줄)

#### Changed
- 문제 수 80개 → 150개로 확장
- 카테고리 수 8개 → 17개로 확장
- AI 튜터 시스템 프롬프트 개선

#### Fixed
- enum 대소문자 불일치 문제 해결 (difficulty, category)
- 시딩 스크립트 enum 값 대문자로 수정
